# Quickstart

```bash
# 1. Clone
$ git clone https://github.com/your-org/iceOS.git
$ cd iceOS

# 2. Install
$ poetry install --with dev

# 3. Run API
$ poetry run uvicorn ice_api.main:app --reload

# 4. Open browser
http://localhost:8000/docs
```
